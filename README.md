# Tamil Nadu Election Analysis (2021–2026)

## Project Overview
This project analyses political shifts between 2021 and 2026 Tamil Nadu Assembly elections.

Research questions:
1. Which regions showed highest political volatility?
2. How many constituencies changed winning parties?
3. Did elections become more competitive?

## Tools
- PostgreSQL
- Tableau
- Canva

## Files
- SQL analysis
- Dashboard
- Stakeholder deck

## Key Findings
- Regional volatility increased in selected regions
- Several constituencies changed parties
- Average victory margin fell from 22,871 to 16,784 votes (~27% decrease)

## Reproduction Steps
1. Import CSV files into PostgreSQL
2. Run SQL queries
3. Export outputs to Tableau
4. Build dashboard
5. Create presentation
